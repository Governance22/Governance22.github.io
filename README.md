# 大白斑蝶與爬森藤植物的互動關係

大白斑蝶為蛺蝶科斑蝶亞科的一種，主要分布地區為東南亞、澳大利亞北部以及台灣南部

## 大白斑蝶的分類位階：

* 域 ： 真核域 Eukarya

* 界 ： 動物界 Animalia

* 門 ： 節肢動物門 Arthropoda

* 綱 ： 昆蟲綱 Insecta

* 目 ： 鱗翅目 Lepidoptera

* 科 ： 蛺蝶科 Nymphalidae

* 屬 ： 帛斑蝶屬 Idea

* 種 ： 大白斑蝶 Idea leuconoe
